# 🌸 MimiEats - Food Delivery Web Application

MimiEats is a simple food delivery web application designed with a sweet and user-friendly interface. This project was developed for the **Web Application Development (WAD)** course assignment.

## ✨ Key Features
- **Buyer Panel**: Search for food, adjust order quantities (+/-), and send orders directly to the seller.
- **Seller Panel**: Receive real-time orders, reply to buyer messages, and a feature to clear chat history.
- **Quick Search**: Instant food recommendations using filter chips (Seblak, Mie Ayam, etc.).
- **Real-time Chat**: Two-way communication between buyer and seller powered by a MySQL database.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3 (Animations & Flexbox), JavaScript.
- **Backend**: PHP.
- **Database**: MySQL.
- **Icons**: Phosphor Icons.

## 🚀 How to Run
1. Download or Clone this repository.
2. Place the project folder into your `htdocs` directory (if using XAMPP).
3. Import the `mimieats` database via phpMyAdmin.
4. Access `localhost/mimieats/index.html` for the Buyer view and `seller.php` for the Seller view.
